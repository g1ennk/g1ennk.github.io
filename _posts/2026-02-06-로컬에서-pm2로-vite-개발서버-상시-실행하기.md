## 문제

요즘 flowmate frontend 작업을 하면서
매번 IDE나 로컬 터미널에서 `npm run dev`로 서버를 띄우는 게 은근히 귀찮았다.

- 터미널 닫으면 서버 꺼지고
- 재부팅하면 다시 실행해야 하고
- 작업 시작할 때마다 반복

개발 서버를 그냥 항상 켜두고 싶었다.

---

## 해결: pm2로 Vite dev 서버 상시 실행

pm2는 Node 기반 앱을 백그라운드에서 실행 및 관리해주는 도구로,
이전에 NEMO 프로젝트 클라우드 작업할 때도 도커 도입 전에 be, fe를 서버에서 실행했던 기억이 있다.

과정은 굉장히 간단하다.
먼저 pm2를 설치하고, pm2로 실행하면 끝이다.

```bash
#설치 및 확인
npm install -g pm2
pm2 -v

# pm2 실행
pm2 start npm --name vite-dev -- run dev -- --port 5173 --host
# --name vite-dev -> pm2에서 관리할 이름
# --port 5173 -> 포트 지정
# --host -> 외부 접속 for 모바일

# 자동 실행
pm2 startup # 시스템 서비스 등록
pm2 save # 현재 상태 저장

# 기타 (서버 중지 및 삭제)
pm2 stop vite-dev
pm2 delete vite-dev
pm2 stop all
pm2 delete all
```

## 결과

- 개발 서버를 매번 켤 필요 없어짐
- 오랜만에 pm2 리마인드
